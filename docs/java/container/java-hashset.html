HashSet

使用

public static void main(String[] args) {
    HashSet<Stu> hashSet = new HashSet<Stu>();

    hashSet.add(new Stu("a", 21));
    hashSet.add(new Stu("b", 20));
    hashSet.add(new Stu("c", 28));
    hashSet.add(new Stu("d", 23));
    hashSet.add(new Stu("e", 25));
    Stu stu = new Stu("e", 25);
    hashSet.add(stu);
    hashSet.add(stu);
    hashSet.add(null);//允许null值

    Iterator<Stu> stuIterator = hashSet.iterator();
    while (stuIterator.hasNext()){
        System.out.println(stuIterator.next());
    }

}

static class Stu implements Comparable {
    private String name;
    private Integer age;
    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
    public Stu(String name, Integer age) {
        this.name = name;
        this.age = age;
    }
    @Override
    public int compareTo(Object o) {
        return this.age.compareTo(((Stu) o).age);
    }
    @Override
    public String toString() {
        return this.name + ": " + this.age;
    }
}

上面的例子会输出几个e: 25?

  • 为什么不是三个：set是集合，集合中元素是不重复的。hashSet.add(stu); hashSet.add(stu); 这个虽然是add了两次，但只会存入一个
  • 为什么是两次：两次new Stu("e", 25)并不是一个对象，HashMap中判断key是否相等是通过equals来的

new HashSet

public HashSet() {
    map = new HashMap<>();
}
public HashSet(Collection<? extends E> c) {
    map = new HashMap<>(Math.max((int) (c.size()/.75f) + 1, 16));
    addAll(c);
}
public HashSet(int initialCapacity, float loadFactor) {
    map = new HashMap<>(initialCapacity, loadFactor);
}
public HashSet(int initialCapacity) {
    map = new HashMap<>(initialCapacity);
}

HashSet(int initialCapacity, float loadFactor, boolean dummy) {
    map = new LinkedHashMap<>(initialCapacity, loadFactor);
}

HashSet是基于HashMap实现的。注意其中也有一个是基于LinkedHashMap的，这个是私有方式，是提供给LinkendHashSet使用的。
